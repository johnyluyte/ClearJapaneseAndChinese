Welcome!

感謝您安裝 Clear Japanese and Chinese chrome extension。

中文使用者長期飽受破碎字型之苦，雖然網路上有一些解決方案，但大多需要修改特殊檔案，令一般使用者怯步。

ClearJC 除了可以一鍵美化字體外，也提供簡易的選單化設定，讓所有使用者都能輕鬆獲得最好的視覺體驗。

接下來，我們就來一步步完成基本設定(大約5分鐘內)。

安裝核心字體

ClearJC 主要支援兩類型字體，分別是微軟的 微軟雅黑體(Microsoft YaHei)(//TODO 微軟正黑體以及日文字體哩) 以及 Google 和 Adobe 合作推出的 思源黑體(Noto Sans T Chinese) ，使用者必須安裝以上至少一種字型，才能獲得最佳字體美化功能。

  • 微軟雅黑體：為 Windows Vista, Windows 7 的預設字體，Windows 使用者不必另外安裝。
  • 思源黑體：可至 官方網站 免費下載，或參考電腦玩物的 教學文。

    • 若您是 Windows 系統的使用者，建議先嘗試 ClearJC 搭配 微軟雅黑體 或 微軟正黑體，如不滿意再下載 思源黑體。

    若您沒安裝某些字體(e.g.思源黑體)，則此外掛內該字體(e.g.思源黑體)的選項都不會產生作用。

    ※ 若您希望 ClearJC 支援其他字體(開源、免費為佳)，請至 github 提出建議，謝謝。

    常見問題

    Q: 我選擇了欲套用的字體，並且按下套用，但是沒有任何效果?
    A:
    1. 請確定您的電腦裡有安裝該字體。
    2. 請設定 directwrite。請參考這裡設定。
    若依舊沒有效果，請打開 Chrome 的 console 、記錄錯誤訊息，並回報至 johnyluyte(dot)gmail(dot)com

    Q: 如何讓 Clear J&C 自動套用在我瀏覽的每一個頁面上?
    A: 請開啟 Clear J&C，點選"其他設定"，勾選 自動套用在所有頁面。
    如此 Clear J&C 便會將您設定的字體套用在之後新開啟的每一個頁面上。

    Q: 我只想在某些網頁套用 微軟雅黑體 字型，其他網頁都用預設，可以嗎?
    A: 現在不行，但我會在之後版本中加入這個功能，預計會用 Regex 表示 domain 的方式來實作。
    Q: 如何將網站加入例外 A: 等 Regex 寫好後，會教你
