Welcome!

感謝您安裝 ClearCJ chrome extension。

中文使用者長期飽受破碎字型之苦，雖然網路上有一些解決方案，但大多需要修改特殊檔案，令一般使用者怯步。

ClearCJ 除了可以一鍵美化字體外，也提供簡易的選單化設定，讓所有使用者都能輕鬆獲得最好的視覺體驗。

安裝核心字體

ClearCJ 主要支援兩類型字體，分別是微軟的 微軟雅黑體(Microsoft YaHei)(TODO 微軟正黑體以及日文字體哩) 以及 Google 和 Adobe 合作推出的 思源黑體(Noto Sans T Chinese) ，使用者必須安裝以上至少一種字型，才能獲得最佳字體美化功能。

  • 微軟雅黑體：為 Windows Vista, Windows 7 的預設字體，Windows 使用者不必另外安裝。
  • 思源黑體：可至 官方網站 免費下載，或參考電腦玩物的 教學文。

    • 若您是 Windows 系統的使用者，建議先嘗試 ClearCJ 搭配 微軟雅黑體 或 微軟正黑體，如不滿意再下載 思源黑體。

    若您沒安裝某些字體(e.g.思源黑體)，則此外掛內該字體(e.g.思源黑體)的選項都不會產生作用。

    ※ 若您希望 ClearCJ 支援其他字體(開源、免費為佳)，請至 github 提出建議，謝謝。
